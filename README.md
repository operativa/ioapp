Para levantar la aplicacion debe descargar este repositorio en su pc, ir hacia la carpeta donde descargo el repositorio, abrir una ventana cmd y ejecutar el comando npm install para bajar todas las dependencias necesarias y luego de que las descargas finalicen ejecutar el comando npm start (en la ventana cmd) en la misma carpeta del proyecto, esto levantara el servidor y podra acceder a la app poniendo en el browser la direccion localhost:3000
